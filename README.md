# 🤖 Gemini AI Chatbot 

🚀 A ChatGPT-like AI chatbot built using Streamlit and Google Gemini API with conversational memory and a modern UI.

🔗 **Live Demo:** https://geminiaichatapp.streamlit.app

---

## ✨ Key Highlights

- 💬 ChatGPT-style chat interface
- 🧠 Context-aware conversation (memory)
- ⚡ Fast real-time responses
- 🔐 Secure API key handling
- 🌐 Deployed on Streamlit Cloud
- 🧹 Clear chat + session handling

---

## 🧠 Problem Solved

Most basic chatbot apps don’t remember context and feel disconnected.  
This project solves that by implementing **session-based memory**, making conversations natural and continuous.

---

## 🛠️ Tech Stack

| Technology | Usage |
|----------|------|
| Python | Core logic |
| Streamlit | UI |
| Gemini API | AI responses |

---

## 🏗️ Architecture

User Input → Streamlit UI → Session Memory → Gemini API → Response → UI Display
